Вам дана последовательность строк.

Выведите строки, содержащие две буквы "**z**﻿", между которыми ровно три символа.

---

**Sample Input**:

```
zabcz
zzz
zzxzz
zz
zxz
zzxzxxz
```

**Sample Output**:

```
zabcz
zzxzz
```

---

# Solution / Решение

```
import re
import sys
pattern = r'(z.{3}z)'
for line in sys.stdin:
    line = line.rstrip()
    if re.search(pattern, line):
        print(line)
```
