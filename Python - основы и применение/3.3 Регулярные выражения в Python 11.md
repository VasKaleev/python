Вам дана последовательность строк.

Выведите строки, содержащие слово, состоящее из двух одинаковых частей (тандемный повтор).

---

**Sample Input**:

```
blabla is a tandem repetition
123123 is good too
go go
aaa
```

**Sample Output**:

```
blabla is a tandem repetition
123123 is good too
```

# Solution / Решение

```
import re
import sys
pattern = r'\b(.+)\1\b'
for line in sys.stdin:
    line = line.rstrip()
    if re.search(pattern, line):
        print(line)
```
